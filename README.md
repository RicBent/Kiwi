# Kiwi
## 지능형 한국어 형태소 분석기(Korean Intelligent Word Identifier)

### 개요
Kiwi는 C++기반의 한국어 형태소 분석기 라이브러리입니다. 
입력한 단어나 문장을 세종 품사 태그에 따라 분석하고 그 태그를 붙여줍니다. 
현재는 개발 초기 단계로 분석기는 약 80%의 정확도로 해당 텍스트를 분석해 낼 수 있습니다.
